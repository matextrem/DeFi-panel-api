post /transactions/borrow/{token}
borrow tokens (borrowTokens)
By passing in the appropriate options, you can borrow tokens thru the different protocols

Path parameters

token (required)
Path Parameter — pass a token name

Query parameters

protocol (required)
Query Parameter — pass a protocol name
value (required)
Query Parameter — pass a token value

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

get /rates/{token}
get rates (getRates)
By passing in the appropriate options, you can get the rates for the different protocols

Path parameters

token (required)
Path Parameter — pass a token name

Query parameters

protocol (required)
Query Parameter — pass a protocol name

Return type

RateItem

Example data

Content-Type: application/json
{
  "protocol" : "compound",
  "rates" : {
    "borrow_rate" : "10.93",
    "supply_rate" : "17.6"
  },
  "token" : "DAI"
}
